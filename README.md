# Trabajo Práctio Final Curso de Python

## Descripción

Repositorio con dos trabajos prácticos finales sobre el curso de Python de la UTN.BA

## Autor

Cecilia Caimi

## Contenido

1. Encuesta de Hogares
Trabajo realizado sobre los archivos obtenidos en https://data.buenosaires.gob.ar/dataset/encuesta-anual-hogares para los años 2016, 2017, 2018 y 2019.
   - Archivo .zip con una carpeta principal
   - Carpeta principal con archivo de consigna y subcarpetas: data y notebooks
   - Subcarpeta data/raw con archivo .csv como dataset
   - Subcarpeta notebooks con archivos.ipynb

### Instrucciones de ejecución

1. Descargar el archivo zip a una carpeta local
2. Descomprimir el archivo
3. Utilizar Jupyterlab para abrir las notebooks
4. Ejecutar las notebooks en el siguiente orden:
   - 01_data_processing.ipynb
   - 02_data_discovery.ipynb
   - 03_EDA.ipynb

2. Ventas de Amazon

   - Carpeta principal con archivo de consigna y subcarpetas: data y notebooks
   - Subcarpeta data con archivo .csv como dataset
